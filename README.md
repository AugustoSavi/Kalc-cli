- [x] O usuário pode usar todas as operações aritméticas básicas como (adição, subtração, multiplicação e divisão)
- [ ] ** Potência de ** e ** Raiz quadrada de ** operação.

## Modo de uso

   Utilize kalc e passe a expressão entre aspas 

```bash
 ~ $ kalc "2+2"
4

 ~ $ kalc "2*2"
4

 ~ $ kalc "(5/2.2)*2.05"
4.659090909090908

 ~ $ kalc "(5/2.2)*2"
4.545454545454545

```
